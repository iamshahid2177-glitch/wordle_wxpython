# wordle_wxpython
# Wordle Game using wxPython

This project is a GUI-based Wordle game developed using Python and wxPython.

## Features
- 5-letter word guessing game
- 6 attempts per game
- Color-coded feedback (Green, Gold, Gray)
- Reset and Help functionality
- Event-driven GUI using wxPython

## Technologies Used
- Python 3
- wxPython

## How to Run
1. Install wxPython:
   pip install wxpython
2. Run the program:
   python wordle_main.py
